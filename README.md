# Basic Docker

A basic Docker website build with NGINX.

- Docker Build: `docker build --tag website .`
- Docker Run: `docker run --publish 80:80 website`
- Docker Compose: `docker compose up`
- Docker Compose Rebuild: `docker-compose up --build`
